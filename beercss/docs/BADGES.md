## Badges

### Common Badges

Styles: default, primary, secondary, tertiary

Alignment badge 'span': default, top left, bottom left, bottom right

### Default Style and Default Alignment badge 'span'

#### Examples:

```html
<a>
  <i>home</i>
  <span class="badge min"></span>
</a>
```

```html
<a>
  <i>home</i>
  <span class="badge">1</span>
</a>
```
