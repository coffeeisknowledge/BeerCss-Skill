## Spaces

space type: 'small-space', 'medium-space', 'large-space'

Default css:
space type: 'small-space'

### Example 'small-space'

```html
<div>
  <div>Some text here</div>
  <div class="space"></div>
  <div>Some text here</div>
  <div class="space"></div>
  <div>Some text here</div>
</div>
```

### Example 'medium-space', 'large-space'

```html
<div>
  <div>Some text here</div>
  <div class="medium-space"></div>
  <div>Some text here</div>
  <div class="medium-space"></div>
  <div>Some text here</div>
</div>
```
