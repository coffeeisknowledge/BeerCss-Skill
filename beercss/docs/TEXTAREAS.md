## Textarea

Multiple Styles: border, round, fill
Rule: These styles can be used together, individually, or simply not used.

Sizes: small, medium, large, extra

**Default CSS elements**
- size: medium

Rule: The default CSS elements do not need to be added to the 'class'

### Default Examples

```html
<div class="field border">
  <textarea></textarea>
</div>
```

```html
<div class="field border">
  <textarea></textarea>
  <output>Helper text</output>
</div>
```

```html
<div class="field invalid border">
  <textarea></textarea>
  <output class="invalid">Error text</output>
</div>
```

```html
<div class="field label border">
  <textarea></textarea>
  <label>Label</label>
</div>
```

```html
<div class="field label border">
  <textarea></textarea>
  <label>Label</label>
  <output>Helper text</output>
</div>
```

```html
<div class="field label invalid border">
  <textarea></textarea>
  <label>Label</label>
  <output class="invalid">Error text</output>
</div>
```

```html
<div class="field label prefix border">
  <i>search</i>
  <textarea></textarea>
  <label>Label</label>
</div>
```

```html
<div class="field label suffix border">
  <textarea></textarea>
  <label>Label</label>
  <i>search</i>
</div>
```

```html
<div class="field label prefix suffix border">
  <i>search</i>
  <textarea></textarea>
  <label>Label</label>
  <i>search</i>
</div>
```
