## Responsive

A button that automatically detects the type of device 

Example:

```html
<nav>
  <button class="s circle extra">
    <i>phone_android</i>
  </button>
  <button class="m l border circle extra">
    <i>phone_android</i>
  </button>
  <button class="m circle extra">
    <i>tablet_android</i>
  </button>
  <button class="s l border circle extra">
    <i>tablet_android</i>
  </button>
  <button class="l circle extra">
    <i>laptop_windows</i>
  </button>
  <button class="s m border circle extra">
    <i>laptop_windows</i>
  </button>
</nav>
```
